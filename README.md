# Cuenta Regresiva para el Gran Día 🎉

Esta aplicación web muestra una cuenta regresiva para el 3 de Marzo de 2026, con efectos visuales interactivos para celebrar la ocasión.

## 🚀 Características
- **Cuenta Regresiva Precisa**: Muestra días, horas, minutos y segundos restantes.
- **Efectos Visuales**: Globos flotantes y confeti animados.
- **Interactivo**: Botón de "Celebrar" que lanza más confeti y emojis.
- **Diseño Moderno**: Uso de Tailwind CSS para un estilo limpio y responsivo.

## 🛠️ Instalación y Uso Local

1.  **Clona o descarga este repositorio.**
2.  **Abre el archivo `index.html`** directamente en tu navegador (doble clic).
    - O usa un servidor local simple: `python -m http.server` y visita `http://localhost:8000`.

## 🌐 Despliegue en Vercel

Esta aplicación es completamente estática, por lo que es muy fácil de desplegar en Vercel.

1.  **Crea una cuenta en [Vercel](https://vercel.com/)** si no tienes una.
2.  **Instala Vercel CLI** (opcional) o conecta tu repositorio de GitHub/GitLab/Bitbucket.
3.  **Nuevo Proyecto**: Selecciona este repositorio.
4.  **Configuración de Build**: Vercel detectará automáticamente que es un sitio estático. No necesitas cambiar nada.
5.  **Desplegar**: Haz clic en "Deploy". ¡Listo!

## 📦 Tecnologías
- HTML5
- CSS3 (Tailwind CSS vía CDN)
- JavaScript (Vanilla)
- canvas-confetti (Librería para efectos de confeti)
